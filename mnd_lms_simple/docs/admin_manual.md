# MND LMS 운영자 매뉴얼

## ? 매뉴얼 개요

### 대상자
- 시스템 관리자
- 교육 담당자
- 강의 관리자
- 기술 지원팀

### 매뉴얼 목적
본 매뉴얼은 MND LMS(국방부 학습관리시스템)의 운영자를 위한 완전한 사용 가이드입니다. 시스템 관리, 사용자 관리, 강의 관리, 문제 해결 등 운영에 필요한 모든 정보를 제공합니다.

## ? 시스템 시작하기

### 1.1 관리자 계정 로그인

1. **로그인 페이지 접속**
   - URL: `https://your-domain.com/login/`
   - 기본 관리자 계정: `admin` / `yang1123!`

2. **로그인 과정**
   ```
   1) 사용자 ID 입력: admin
   2) 비밀번호 입력: yang1123!
   3) "로그인" 버튼 클릭
   4) 관리자 권한 확인 후 리디렉션
   ```

3. **첫 로그인 후 할 일**
   - 기본 비밀번호 변경 (보안 강화)
   - 시스템 설정 확인
   - 초기 데이터 확인

### 1.2 관리자 대시보드 접근

- **URL**: `https://your-domain.com/admin-dashboard/`
- **접근 방법**: 
  1. 로그인 후 상단 네비게이션에서 "관리자 대시보드" 클릭
  2. 또는 직접 URL 입력

## ? 사용자 관리

### 2.1 사용자 목록 조회

**접근 경로**: 관리자 대시보드 → 사용자 관리

**주요 기능**:
- 전체 사용자 목록 조회
- 검색 및 필터링
- 사용자 상태 확인
- 개인정보 마스킹 처리

**화면 구성**:
```
┌─────────────────────────────────────────┐
│ 검색바: [사용자명/이메일 검색] [등급필터]   │
├─────────────────────────────────────────┤
│ 사용자명 | 이름 | 이메일 | 등급 | 상태    │
├─────────────────────────────────────────┤
│ admin   | 관리자| ad***@**.kr | A | 활성 │
│ user001 | 김**  | us***@**.kr | B | 활성 │
└─────────────────────────────────────────┘
```

### 2.2 사용자 상세 정보 관리

**기능**:
- 사용자 기본 정보 조회/수정
- 로그인 기록 확인
- 학습 진도 현황
- 권한 설정

**수행 절차**:
```
1. 사용자 목록에서 "상세" 버튼 클릭
2. 사용자 정보 확인
3. 필요시 정보 수정
4. "저장" 버튼으로 변경사항 저장
```

### 2.3 사용자 계정 관리

#### 계정 활성화/비활성화
```
활성화: 사용자 상세페이지 → "계정 활성화" 버튼
비활성화: 사용자 상세페이지 → "계정 비활성화" 버튼
```

#### 비밀번호 재설정
```
1. 사용자 상세페이지 접근
2. "비밀번호 재설정" 버튼 클릭
3. 임시 비밀번호 생성 확인
4. 사용자에게 새 비밀번호 전달
```

## ? 강의 관리

### 3.1 강의 목록 관리

**접근 경로**: 관리자 대시보드 → 강의 관리

**주요 기능**:
- 강의 목록 조회
- 강의 상태 관리 (게시됨/비공개/준비중)
- 수강생 현황 확인
- 강의 검색 및 필터링

### 3.2 새 강의 등록

**등록 절차**:
```
1. 강의 관리 페이지 → "새 강의 등록" 버튼
2. 강의 기본 정보 입력:
   - 강의명*
   - 강의 설명*
   - 강사명*
   - 카테고리 선택*
   - 난이도 (초급/중급/고급)
   - 수강 기간 (주 단위)
   - 최대 수강생 수
   - 수강료 (기본값: 무료)
3. 강의 일정 설정:
   - 개강일
   - 종강일
4. 추가 옵션:
   - 추천 강의 여부
   - 강의 상태 (기본: 준비중)
5. "등록" 버튼으로 저장
```

### 3.3 강의 수정 및 삭제

**수정 절차**:
```
1. 강의 목록에서 수정할 강의 선택
2. "편집" 버튼 클릭
3. 정보 수정 후 "저장"
```

**삭제 주의사항**:
- 수강생이 있는 강의는 삭제 불가
- 삭제 전 관련 데이터 백업 권장
- 삭제는 복구 불가능

### 3.4 수강 관리

**수강신청 승인**:
```
1. 강의 상세페이지 → "수강신청 관리"
2. 대기 중인 신청 목록 확인
3. 개별 승인/거부 또는 일괄 처리
```

**수강생 관리**:
```
- 수강생 목록 조회
- 개별 수강 취소
- 진도율 확인
- 성적 관리
```

### 7.3 백업 및 복구

**백업 절차**:
```
1. 데이터베이스 백업
   python manage.py dumpdata > backup_YYYYMMDD.json

2. 미디어 파일 백업
   tar -czf media_backup_YYYYMMDD.tar.gz media/

3. 설정 파일 백업
   cp .env .env.backup_YYYYMMDD
```

**복구 절차**:
```
1. 서비스 중단
2. 데이터베이스 복구
   python manage.py loaddata backup_YYYYMMDD.json
3. 미디어 파일 복구
4. 서비스 재시작
5. 정상 작동 확인
```

## ? 문제 해결

### 8.1 일반적인 문제

**로그인 문제**:
```
증상: 관리자 로그인 실패
원인: 비밀번호 변경, 계정 잠금, 세션 만료
해결: 
1. 비밀번호 확인
2. 쿠키/캐시 삭제
3. 필요시 비밀번호 재설정
```

**성능 저하**:
```
증상: 페이지 로딩 속도 저하
원인: 데이터베이스 부하, 메모리 부족
해결:
1. 데이터베이스 최적화
2. 캐시 설정 확인
3. 서버 리소스 점검
```

### 8.2 에러 코드 및 해결방법

**500 Internal Server Error**:
- 로그 파일 확인: `tail -f django.log`
- 설정 파일 점검
- 데이터베이스 연결 확인

**404 Not Found**:
- URL 패턴 확인
- 템플릿 파일 존재 여부
- static 파일 경로 확인
