# MND LMS 사용자 매뉴얼

## ? 사용자 가이드 개요

### 환영합니다!
MND LMS(국방부 학습관리시스템)에 오신 것을 환영합니다. 본 매뉴얼은 일반 사용자를 위한 완전한 사용 가이드입니다.

### 대상자
- 교육 수강생
- 군 간부
- 민간 교육생
- 일반 이용자

### 주요 기능
- 온라인 강의 수강
- 학습 진도 관리
- 커뮤니티 참여
- 개인 학습 기록 관리

## ? 시작하기

### 1.1 회원가입

**회원가입 절차**:
```
1. 메인 페이지 접속: https://your-domain.com
2. "회원가입" 버튼 클릭
3. 회원가입 양식 작성:
   - 사용자 ID* (영문+숫자, 6-20자)
   - 비밀번호* (8자 이상, 영문+숫자+특수문자)
   - 비밀번호 확인*
   - 이름* (실명)
   - 이메일* (본인 인증용)
   - 소속 (부대명 또는 기관명)
   - 전화번호
4. 개인정보 처리 동의
5. "가입하기" 버튼 클릭
6. 이메일 인증 완료
```

**아이디/비밀번호 규칙**:
- **아이디**: 영문 또는 영문+숫자 조합, 6-20자
- **비밀번호**: 최소 8자, 영문+숫자+특수문자 포함 권장

### 1.2 로그인 및 로그아웃

**로그인 방법**:
```
1. 메인 페이지에서 "로그인" 클릭
2. 사용자 ID와 비밀번호 입력
3. "로그인" 버튼 클릭
4. 성공 시 메인 대시보드로 이동
```

**로그아웃**:
- 우측 상단 프로필 → "로그아웃" 클릭

**로그인 문제 해결**:
- 비밀번호 오류: "비밀번호 찾기" 이용
- 계정 잠금: 관리자 문의 (admin@mnd-lms.go.kr)

## ? 강의 수강하기

### 3.1 강의 찾기

**검색 방법**:
```
1. 상단 검색바에 키워드 입력
2. 검색 결과 확인
3. 관심 강의 클릭하여 상세 정보 확인
```

**카테고리별 탐색**:
- **전술/전략**: 군사 전술, 전략 이론
- **리더십**: 지휘관 역량, 팀 관리
- **기술교육**: IT, 통신, 장비 운용
- **어학**: 영어, 기타 외국어
- **교양**: 역사, 문화, 인문학

**강의 정보 확인**:

### 3.3 수강신청 및 결제

**수강신청 절차**:
```
1. 강의 상세페이지에서 "수강신청" 클릭
2. 수강 조건 확인:
   - 선수 강의 이수 여부
   - 수강 자격 요건
   - 수강료 확인
3. 수강신청서 작성
4. 결제 (유료 강의의 경우)
5. 신청 완료 확인
```

**결제 방법** (유료 강의):
- 신용카드
- 계좌이체
- 가상계좌
- 국방통합카드

**수료증 발급**:

## ? 도움말 및 문제해결

### 8.1 자주 묻는 질문 (FAQ)

**Q: 강의 동영상이 재생되지 않아요**
```
A: 다음 사항을 확인해보세요:
1. 인터넷 연결 상태 확인
2. 브라우저 새로고침 (Ctrl+F5)
3. 다른 브라우저로 시도
4. 브라우저 쿠키/캐시 삭제
5. 문제 지속시 관리자 문의
```

**Q: 수강신청이 안 됩니다**
```
A: 가능한 원인:
- 수강 정원 초과
- 선수 강의 미이수
- 결제 문제 (유료 강의)
- 수강 자격 미달
→ 강의 상세정보에서 조건 확인
```

**Q: 진도가 저장되지 않아요**
```
A: 해결 방법:
1. 로그인 상태 확인
2. 동영상 충분히 시청 (최소 80%)
3. 다음 차시로 넘어가기 전 잠시 대기
4. 브라우저 쿠키 허용 확인
```
