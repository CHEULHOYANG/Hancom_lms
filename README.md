# 한컴 국방부 교육 시스템 (Hancom Defense Ministry LMS)

한컴이 개발한 국방부 전용 학습 관리 시스템(Learning Management System)입니다.

## 프로젝트 개요

본 시스템은 대한민국 국방부의 교육 요구사항에 맞춰 설계된 전문 교육 플랫폼으로, 국방 분야의 체계적인 교육 및 훈련을 지원합니다.

## 주요 기능

- **교육 과정 관리**: 다양한 국방 관련 교육 과정 제공
- **학습 진도 추적**: 실시간 학습 현황 모니터링
- **보안 강화**: 국방 보안 기준에 맞는 강화된 보안 시스템
- **온·오프라인 혼합 교육**: 다양한 교육 방식 지원
- **성과 분석**: 교육 효과 분석 및 리포팅

## 제공 교육 과정

1. **정보보안 기초** (4주 과정)
   - 국방 정보보안의 기본 원칙과 실무

2. **사이버 방어** (6주 과정)
   - 사이버 위협 대응 및 방어 전략

3. **군사 지휘체계** (8주 과정)
   - 현대 군사 지휘체계의 이해와 운용

4. **국방 IT 시스템** (5주 과정)
   - 국방 정보시스템 구축 및 관리

## 시스템 구조

```
Hancom_lms/
```

## 시작하기

### 설치 방법

1. 저장소 클론:
```bash
git clone https://github.com/CHEULHOYANG/Hancom_lms.git
cd Hancom_lms
```

2. 웹 브라우저에서 `index.html` 파일 열기

### 로컬 서버 실행 (선택사항)

Python 3를 사용하는 경우:
```bash
python -m http.server 8000
```

그런 다음 브라우저에서 `http://localhost:8000` 접속

## 보안 요구사항

- AES-256 암호화
- 2단계 인증 지원
- 세션 타임아웃: 30분
- 강력한 비밀번호 정책
- IP 화이트리스트 지원

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript
- **데이터**: JSON
- **보안**: 국방부 보안 기준 준수
