## 시작하기

### 시스템 접속
1. 웹 브라우저를 실행합니다
2. 시스템 URL에 접속합니다
3. 로그인 화면이 표시됩니다

### 권장 브라우저
- Google Chrome (최신 버전)
- Mozilla Firefox (최신 버전)
- Microsoft Edge (최신 버전)
- Safari (최신 버전)

## 로그인

### 로그인 절차

1. **사용자 ID 입력**
   - 학생: 군번 입력 (예: 20250001)
   - 교관: 교관ID 입력 (예: INS-001)
   - 관리자: 관리자ID 입력

2. **비밀번호 입력**
   - 개인 비밀번호 입력
   - 최초 로그인 시 비밀번호 변경 권장

3. **사용자 구분 선택**
   - 학생 (병사)
   - 교관
   - 관리자

4. **로그인 버튼 클릭**

### 데모 모드
현재 시스템은 데모 모드로 운영됩니다:
- 모든 ID/비밀번호 조합이 허용됩니다
- 데이터는 브라우저에 임시 저장됩니다
- 실제 운영 환경에서는 정식 인증이 필요합니다

### 로그인 문제 해결
- **아이디/비밀번호 오류**: 입력 정보 재확인
- **사용자 구분 미선택**: 드롭다운에서 반드시 선택
- **페이지 로딩 안됨**: 인터넷 연결 확인

## 대시보드

### 개요
대시보드는 시스템의 메인 화면으로, 전체 현황을 한눈에 파악할 수 있습니다.

### 주요 통계
1. **진행 중인 교육**: 현재 활성화된 교육 과정 수
2. **등록 학생**: 시스템에 등록된 총 학생 수
3. **완료율**: 전체 교육 과정의 평균 완료율
4. **교관 현황**: 배정된 교관 수

### 최근 활동
- 시스템의 최근 활동 내역 표시
- 신규 등록, 과정 완료, 과정 개설 등의 이벤트 확인
- 실시간 업데이트

### 대시보드 활용 팁
- 정기적으로 통계를 확인하여 교육 진행 상황 모니터링
- 완료율이 낮은 경우 학생 독려 또는 과정 개선 필요
- 최근 활동을 통해 시스템 사용 패턴 파악

## 교육과정

### 교육과정 목록 보기
"교육과정" 메뉴를 클릭하면 다음 정보가 표시됩니다:
- 과정명
- 과정코드
- 과정 설명
- 교육 기간
- 현재 수강생 수

### 제공 과정

1. **군사훈련 기본과정 (MND-001)**
   - 기간: 4주
   - 내용: 기본 군사훈련 및 체력단련
   - 대상: 신병 및 보직 변경 병사

2. **통신보안 교육 (MND-002)**
   - 기간: 2주
   - 내용: 군사통신 보안 및 암호화 기술
   - 대상: 통신병과 관련 보직

3. **전술훈련 교육 (MND-003)**
   - 기간: 6주
   - 내용: 전술적 사고 및 작전 수행
   - 대상: 간부 및 고참 병사

4. **응급처치 및 구급 (MND-004)**
   - 기간: 3주
   - 내용: 전장 응급처치 및 구급
   - 대상: 의무병 및 전 병사

5. **사이버전 대응 (MND-005)**
   - 기간: 8주
   - 내용: 사이버 공격 및 방어
   - 대상: 사이버 작전 요원

### 신규 과정 추가 (교관/관리자)
1. "신규 과정 추가" 버튼 클릭
2. 과정 정보 입력
3. 저장

### 과정 상세 정보
"상세보기" 버튼을 클릭하여:
- 커리큘럼 확인
- 교육 자료 다운로드
- 수강생 명단 확인
- 진도 현황 파악

## 학생 관리

### 학생 목록 조회
학생 관리 섹션에서 다음 정보를 확인할 수 있습니다:
- 군번
- 이름
- 계급
- 소속 부대
- 수강 과정
- 진도율

### 학생 검색
1. 검색창에 이름 또는 군번 입력
2. 실시간으로 결과 필터링
3. 해당 학생 정보 확인

### 신규 학생 등록 (교관/관리자)
1. "학생 추가" 버튼 클릭
2. 필수 정보 입력:
   - 이름
   - 군번
   - 계급
   - 소속 부대
3. 과정 배정
4. 저장

### 학생 진도 관리
- 진도율 바를 통한 시각적 확인
- 진도가 낮은 학생 독려
- 상세 버튼으로 학습 기록 확인

### 학생 정보 수정
1. 해당 학생의 "상세" 버튼 클릭
2. 정보 수정
3. 저장

## 교관 관리

### 교관 목록
교관 관리 섹션에서 확인 가능한 정보:
- 이름 및 계급
- 교관 ID
- 전문 분야
- 담당 과정 수

### 교관 프로필
각 교관 카드에서 확인할 수 있는 정보:
- 기본 정보
- 전문 분야
- 담당 과정 목록
- 교육 실적

### 신규 교관 등록 (관리자)
1. "교관 추가" 버튼 클릭
2. 교관 정보 입력:
   - 이름
   - 계급
   - 교관 ID
   - 전문 분야
3. 과정 배정
4. 저장

### 교관 배정 관리
- 각 과정에 적절한 교관 배정
- 교관 업무 부담 균형 유지
- 전문 분야에 맞는 과정 배정

## 프로필 설정

### 프로필 확인
"프로필" 메뉴에서 확인 가능한 정보:
- 사용자 이름
- 사용자 구분
- 사용자 ID
- 등록일
- 마지막 로그인

### 비밀번호 변경
1. "비밀번호 변경" 버튼 클릭
2. 현재 비밀번호 입력
3. 새 비밀번호 입력 및 확인
4. 저장

### 보안 권장사항
- 정기적인 비밀번호 변경 (3개월마다)
- 복잡한 비밀번호 사용 (영문, 숫자, 특수문자 조합)
- 타인과 비밀번호 공유 금지
- 의심스러운 활동 즉시 보고

## 문제 해결

### 일반적인 문제

**Q: 로그인이 안 됩니다**
A: 
- 사용자 ID와 비밀번호를 확인하세요
- 사용자 구분을 올바르게 선택했는지 확인하세요
- 브라우저 캐시를 삭제해보세요

**Q: 페이지가 로딩되지 않습니다**
A:
- 인터넷 연결을 확인하세요
- 브라우저를 새로고침하세요 (F5)
- 다른 브라우저로 시도해보세요

**Q: 데이터가 저장되지 않습니다**
A:
- 브라우저의 쿠키 및 로컬 스토리지 설정을 확인하세요
- 시크릿/프라이빗 모드가 아닌지 확인하세요
- 저장 버튼을 클릭했는지 확인하세요

**Q: 화면이 깨져 보입니다**
A:
- 브라우저 확대/축소를 100%로 설정하세요
- 최신 브라우저 버전을 사용하세요
- 화면 해상도를 1024x768 이상으로 설정하세요

### 기술 지원

문제가 지속되는 경우:
- 이메일: support@hancom.com
- 전화: 1588-XXXX

2. 관리자에게 즉시 보고
3. 상황 기록 및 스크린샷 캡처
4. 지시가 있을 때까지 대기
